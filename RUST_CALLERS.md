## What this file is

For every Rust crate this repo owns (under `tmp/`), this file lists every **cross-repo** workspace caller: every `use` statement in another `*-kotlin` repo's Rust source that imports a symbol from one of this repo's crates. Symbols are grouped per-crate; codex-kotlin sites are surfaced first per symbol (codex is the priority consumer driving the port queue).

**Cross-repo only.** Imports between crates inside this same repo are omitted — those are intra-repo refactoring concerns, not the cross-project porting concern this file exists to surface.

**Cargo-resolved.** Counts here honor `package = "..."` renames, `workspace = true` inheritance, and `[target.<cfg>.dependencies]`. Stdlib (`std`, `core`, `alloc`) and intra-crate sub-module references are excluded.
